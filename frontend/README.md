# PEPEGAME HUB – Frontend

Ce dossier contiendra l'application web (MVP) :

- Page Hub : affichage du pseudo, des PEPE POINTS et navigation vers les jeux.
- Page TapGame : mini-jeu tap-to-earn PEPE.
- Page FlappyGame : mini-jeu type flappy/runner.
- Page Profile : stats du joueur et historique de points.

Tech prévue pour plus tard :
- React pour les pages.
- Phaser.js pour les jeux.
- Intégration future avec un backend (Firebase) et MetaMask.
